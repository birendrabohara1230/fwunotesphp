End-Term Examination-2079

Theory of Computation (CSIT. 226)

Faculty:Science and Technology (CSIT)
Level: Undergraduate
Semester: Fourth
Full Marks: 100
Time: 3hrs

Candidates are required to give their answers in their own words as far as practicable. The figures in the margin indicate full marks.

Group A

Attempt all questions.(8x3=24)
  1. Define string and length of string with example.
  2. Design DFA over (0,1) that accept string which contains second symbol is 0 and third symbol is 1.
  3. Write down the formal definition of NFA.
  4. DEfine regular expression.
  5. Write down the formal definition of Moore machine.
  6. Describe direct derivation and derivation.
  7. How can you define Chomsky's normal form?
  8. Give the formal definition of Turing machine.
    9. Group B

    Attempt any five questions. (Short questions) (5x8=40)
  9. Explain extended transition function of NFA with example.
  10. Convert the given regular expressoin 1(01+10)*+0(11+101)* into finite automata.
  11. Explain different algebraic rule of regular expression.
  12. Explain yield of parse tree and ambiguous grammar with example.
  13. Define regular grammar. Convert the following grammar into finite automata
    S -> 0B||111C|ε
    B -> 00C|11
    C -> 00B||0D
    D -> 0B|0D
  14. How can you convert CFG into PDA? Convert following CFG into PDA
    S -> aAA
    A -> aS|bS|a
    15. Group C

    Attempt any three questions. (Long questions) (3x12=36)
  15. Minimize the following DFA using table filling method.
  16. Convert following grammar into CNF
    S -> ASB| ε
    A -> aAS|a
    B -> SbS|A|bb
  17. Construct Turing Machine accepting a language of palindrome over {a,b}* with each string of odd length. Explain with example how it accepts.
  18. Define immediate left recursion. How can you remove left recursion from grammar? Remove left recursion from following grammar.
    S -> AA|0
    A -> AAS|0S|1
